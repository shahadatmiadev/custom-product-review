== Description ==

**RevWoo** is a comprehensive review management system designed specifically for WooCommerce stores. It provides a feature-rich alternative to the default WooCommerce review system with advanced filtering, moderation, and customization options.

= Key Features =

* **Advanced Review Form** - Customizable review submission forms with star ratings, titles, descriptions, and file uploads
* **Smart Filtering** - Filter reviews by rating, age range, and verified buyer status
* **Admin Approval System** - Manual or automatic review approval with pending/approved/rejected status management
* **Age Range Tracking** - Collect demographic data with age range selection
* **File Upload Support** - Allow customers to upload images or PDF documents with their reviews
* **Email Notifications** - Get notified when new reviews are submitted
* **Content Moderation** - Bad words filter to automatically reject inappropriate reviews
* **Verified Buyer Badge** - Display verified purchase badges on reviews
* **Complete Style Control** - Customize every aspect of form, review display, and filter appearance
* **Responsive Design** - Mobile-friendly interface that works on all devices
* **AJAX Filtering** - Fast, seamless review filtering without page reloads
* **Load More Pagination** - Smooth loading of additional reviews
* **Custom CSS Support** - Add your own CSS for ultimate design flexibility

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "RevWoo"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin zip file
2. Log in to your WordPress admin panel
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the downloaded zip file and click "Install Now"
5. Activate the plugin after installation

= After Activation =

1. Go to **Product Reviews > Settings** to configure general settings
2. Visit **Product Reviews > Customize Styles** to customize appearance
3. Navigate to any WooCommerce product page to see the review form
4. Manage submitted reviews from **Product Reviews > All Reviews**

== Frequently Asked Questions ==

= Does this plugin work with any WooCommerce theme? =

Yes! RevWoo is designed to work with any properly coded WooCommerce theme.

= Can I customize the review form fields? =

Yes, you can enable/disable file uploads, age range selection, and make email/title fields optional or required through the settings.

= How do I approve or reject reviews? =

Go to Product Reviews > All Reviews in your admin panel. You can approve, reject, or delete reviews with one click. You can also enable auto-approval in settings.

= Can customers upload images with their reviews? =

Yes! Customers can upload JPG, PNG, or PDF files with their reviews. You can enable or disable this feature in the settings.

= How does the filtering system work? =

Customers can filter reviews by star rating, age range, and verified buyer status. The filtering happens instantly using AJAX without page reloads.

= Can I customize the colors and styles? =

Absolutely! The plugin includes a comprehensive style customization panel where you can change colors, borders, padding, and more. You can also add custom CSS.

= Does it support multiple languages? =

Yes, the plugin is translation-ready and includes a .pot file for easy translation.

= Will this slow down my website? =

No! The plugin is optimized for performance and uses AJAX to load reviews efficiently. It only loads necessary resources on product pages.

= Can I set a minimum star rating requirement? =

Yes, you can set a minimum star rating (1-5) in the settings. Reviews below this rating will not be accepted.

= How do I enable email notifications? =

Go to Product Reviews > Settings > Advanced tab and enable email notifications. You can specify the admin email address to receive alerts.

= Can I filter out bad words from reviews? =

Yes! Enable content moderation in the Advanced settings and add a comma-separated list of words to automatically reject reviews containing those terms.

= What happens to reviews when I deactivate the plugin? =

Your reviews are stored safely in the WordPress database. They won't be deleted when you deactivate the plugin, and they'll be available again if you reactivate it.

= Can I export reviews? =

Currently, reviews are stored as a custom post type in WordPress. You can use standard WordPress export tools or database queries to export review data.

= Is this GDPR compliant? =

The plugin collects only the information customers provide (name, email, review text). You should add appropriate privacy notices on your site according to GDPR requirements.

= Can I display reviews on pages other than product pages? =

By default, reviews are displayed on WooCommerce product pages. If you need custom display options, you can use WordPress hooks and filters.
